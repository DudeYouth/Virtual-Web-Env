### Virtual Web Env

+ 申请成功后，打开你的git bash输入：

```bash
	ssh root@123.56.220.42 -p 22 # 请输入自己的端口号
```

+ 输入您的密码

### 使用规则

+ 一个ip只能申请一台虚拟机
+ 最多只能开启5台虚拟机（支持5个用户）

### 端口映射规则

+ 21 		=> [1-5]0021  # ftp服务
+ 22 		=> [1-5]0022  # ssh服务 
+ 23 		=> [1-5]0023  # 自定义服务
+ 80 		=> [1-5]0080  # web服务
+ 3000 		=> [1-5]3000  # 自定义服务(默认node)
+ 3306 		=> [1-5]3306  # mysql服务
+ 8080		=> [1-5]8080  # 自定义服务
+ 8888		=> [1-5]8888  # 自定义服务
